# Vi-SlideS Frontend

React + TypeScript + Vite application for the Vi-SlideS teacher dashboard.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

```
src/
├── components/     # Reusable UI components (Button, Card, Navbar)
├── pages/
│   └── teacher/    # Teacher pages (Dashboard, CreateSession, LiveSession)
│   └── student/    # student pages
├── assets/         # Static assets
├── App.tsx         # Main app with routing
└── index.css       # Global styles and CSS variables
```
